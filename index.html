Bagaimana cara menampilkan foto asli dari folder Google Drive Anda?

Web ini membaca berkas langsung secara cerdas dari folder publik. Anda hanya perlu memastikan:

  1. Ubah hak akses folder Anda di Google Drive menjadi "Siapa saja yang memiliki link dapat melihat" (Public).
  2. Buka menu Pengaturan Folder di kanan atas, matikan "Mode Demo" dan simpan ID Folder Anda.
Urutkan:
Kesalahan Sambungan Silakan periksa kembali ID Folder Google Drive Anda atau ubah status folder menjadi Publik.
